# phonepe-PowerBI-Dashboard
This project presents an interactive Power BI dashboard developed using PhonePe transaction data extracted from transaction statements. The dataset contains transaction records including date, time, transaction type (Debit/Credit), transaction amount, transaction details, transaction ID, and UTR number.
The objective of this dashboard is to transform raw transaction data into meaningful insights and provide a clear understanding of spending patterns, payment behavior, and transaction trends through data visualization.
Dataset Description
The dataset was prepared from PhonePe transaction history and includes:
Transaction Date
Transaction Time
Transaction Details
Transaction Type (Credit / Debit)
Transaction Amount
Transaction ID
UTR Number
Data preprocessing and structuring were performed before importing into Power BI for analysis and dashboard creation.
Dashboard Features
KPI Cards for Total Transaction Amount and Average Transaction Value
Credit vs Debit Transaction Analysis
Day-wise Transaction Distribution
Interactive Slicers for Transaction Type and Day Filtering
Transaction Trend Analysis
Category-wise Transaction Insights
Visual representation of transaction patterns
Key Insights
Identified overall transaction volume and average transaction value
Compared Debit and Credit distributions
Analyzed transaction activity across different days
Visualized transaction behavior using multiple chart types
